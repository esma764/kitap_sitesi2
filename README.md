# 📚 Kitap Öneri Sitesi

Bu proje, **Web Programlama** dersi kapsamında PHP öğrenmek amacıyla geliştirilmiştir.  
Kullanıcılar kitap önerisi yapabilir, kitap listelerini görüntüleyebilir ve diğer kullanıcıların önerilerini inceleyebilir.

---

## 🚀 Özellikler
- Ana sayfa (`index.php`) → Popüler kitaplar, günün kitabı, hızlı erişim
- Kitaplar sayfası (`kitaplar.php`) → PHP dizisi ile kitap listesi
- Öneri sayfası (`oner.php`) → Form ile kitap önerisi, dosyaya kaydetme
- Yorumlar sayfası (`comments.php`) → Dosyadan önerileri listeleme
- Hakkında sayfası (`hakkinda.php`) → Proje tanıtımı
- İletişim sayfası (`contact.php`) → Basit iletişim formu
- CSS (`style.css`) → Modern koyu tema, hover efektleri, kart tasarımı

---

## 🛠️ Kullanılan Teknolojiler
- **HTML** → Sayfa yapısı
- **CSS** → Görsel tasarım
- **PHP** → Dinamik içerik, form işlemleri, dosya kaydetme/okuma
